OBS PROJECTOR STYLE TIMER

1. Open OBS.
2. If OBS already uses your webcam, click Start Virtual Camera.
3. Double-click setup-auto-pause.bat.
4. Pick the number from the preview image that shows the camera feed you want.
5. In OBS, add a Browser Source and choose obs-projector-timer.html as the local file.

Daily use:
Double-click start-helper-hidden.vbs, then open OBS.

Check auto-pause:
Double-click check-status.bat.
You want camera_ok: true.

Controls in OBS Interact:
1-5 = dim to ultra
Arrow Up / Down = brighter / dimmer
Arrow Right / Left = more / less bloom and blur
0 = bathroom break
R = reset timer
