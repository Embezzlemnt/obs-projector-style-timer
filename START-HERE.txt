OBS PROJECTOR STYLE TIMER

1. Double-click START HERE - OBS Projector Timer.bat.
2. Open OBS.
3. Add a Browser Source and choose obs-projector-timer.html as the local file.

That is it. The helper starts quietly with Windows after install.

If OBS already uses your webcam, click Start Virtual Camera in OBS.
The helper finds the camera feed automatically.

To stop the background helper:
Double-click stop-helper.bat.

Controls in OBS Interact:
1-5 = dim to ultra
Arrow Up / Down = brighter / dimmer
Arrow Right / Left = more / less bloom and blur
0 = bathroom break on / off
R = reset timer
